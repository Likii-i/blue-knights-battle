# MBTI Fighters (v1.1)

Current focus: a single ENFP-ish agent with committed intents, adaptive forward planning, mixed emotions, and pressure-induced mistakes in a blank square world.

## Run locally

From this folder:

```sh
python3 -m http.server 5173
```

Then open `http://localhost:5173` in your browser.

## Controls

- Click: place/move fruit
- Shift + Click: place/move the juggernaut (threat)
- `M`: toggle manual control (WASD)
- `D`: toggle debug overlay
- `H`: toggle help overlay
